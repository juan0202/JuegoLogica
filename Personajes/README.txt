Name:
  Battle Axe/Weapons Rework
Version:
  0.2
Description:
  GrumpyDiamond's axe scaled down & reworked into a sprite sheet for
  animating.
Details:
  - Orientation: orthogonal (N/E/S/W)
  - Dimensions:
    - Single strip:
      - 14x14, 16x16
      - Animation: 8 frames (animated preview at 100ms delay)
    - For Cabbit style sprites:
      - 24x32
      - Animation: 12 frames (3 per direction)
  - PNG images use indexed color (smaller files).
  - GIMP sources (.xcf) use RGB color.
